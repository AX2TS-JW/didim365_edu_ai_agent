# 부동산 실거래가 분석 에이전트 메모리

## 사용자 선호

사용자가 관심을 표현한 지역이나 조건은 기억하여 다음 대화에서 활용하세요.
- "내가 관심 있는 지역" → 이전에 자주 조회한 지역 기반 추천
- "지난번에 본 데이터" → 이전 세션 분석 결과 참조

## 부동산 도메인 지식

### 전세가율 판단 기준 (한국부동산원/KB부동산 기준)
- 70% 이상: 위험 (깡통전세 가능성, 매매가 하락 시 보증금 미반환 위험)
- 60~70%: 높음 (매매 전환 적극 고려, 갭투자 주의)
- 40~60%: 보통 (자금 여력에 따라 매매/전세 선택)
- 40% 미만: 낮음 (고가 지역 특성, 전세로 거주 시 자금 효율적)
- 이 기준은 상대 비교(특정 지역 대비)가 아닌 절대 기준으로 적용하세요.

### 투자 판단 시 고려 요소
1. 실거래가 추이 (최근 3~6개월)
2. 전세가율 (매매 vs 전세 갭)
3. 정책/규제 (투기과열지구, 토지거래허가구역)
4. 공급 계획 (재건축/재개발, 신규 분양)
5. 입지 (학군, 교통, 상권)

### 서울 주요 지역 특성
- 강남구: 고가 시장, 재건축 기대감, 학군 프리미엄
- 서초구: 강남 인접, 반포 재건축
- 송파구: 잠실 중심, 대규모 단지
- 마포구: 마포-용산 개발 수혜
- 용산구: 용산 재개발, 대통령실 이전 효과
