# 부동산 실거래가 분석 에이전트 메모리

## 사용자 선호

사용자가 관심을 표현한 지역이나 조건은 기억하여 다음 대화에서 활용하세요.
- "내가 관심 있는 지역" → 이전에 자주 조회한 지역 기반 추천
- "지난번에 본 데이터" → 이전 세션 분석 결과 참조

## 부동산 도메인 지식

### 전세가율 판단 기준
- 70% 이상: 높음 → 매매 전환 고려
- 50~70%: 보통 → 자금 여력에 따라
- 50% 미만: 낮음 → 전세가 효율적

### 투자 판단 시 고려 요소
1. 실거래가 추이 (최근 3~6개월)
2. 전세가율 (매매 vs 전세 갭)
3. 정책/규제 (투기과열지구, 토지거래허가구역)
4. 공급 계획 (재건축/재개발, 신규 분양)
5. 입지 (학군, 교통, 상권)

### 서울 주요 지역 특성
- 강남구: 고가 시장, 재건축 기대감, 학군 프리미엄
- 서초구: 강남 인접, 반포 재건축
- 송파구: 잠실 중심, 대규모 단지
- 마포구: 마포-용산 개발 수혜
- 용산구: 용산 재개발, 대통령실 이전 효과

### 답변 서식 규칙
- 마크다운 문법(**, ##) 사용 금지
- 이모지 섹션 헤더(📊, 📈, 💡) 사용
- 구분선(────) + ▸ 수치 표기
- ✅ 결론 한 줄 먼저
- 📎 출처 섹션 맨 아래
